자바스크립트 엔진 성능이 향상되면서 Node.js로 많은 라이브러리가 생성되고 있습니다. 여러 라이브러리를 설치해서 사용하다 보면 라이브러리마다 필요한 Node 버전이 다른 경우가 간혹 생깁니다. 이럴 때 여러 개의 Node 버전을 관리해주고 현재 실행되는 Node 버전을 바꿀 수 있게 해주는 것이 바로 Node Version Manager, NVM 입니다.
